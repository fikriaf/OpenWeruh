# OpenWeruh Demo

This folder contains scripts or assets used for demonstrating the capabilities of OpenWeruh.

To record a demo:
1. Ensure OpenClaw is running with Webhooks enabled.
2. Start the daemon in verbose mode (or ensure stdout is visible).
3. Open an article containing a false claim (if using `skeptic` mode) or a long Reddit thread (if using `guardian` mode).
4. Watch as the daemon triggers the webhook and OpenClaw responds proactively.
